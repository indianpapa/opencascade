Detailed Description

The class to transfer glTF files. Reads and Writes any glTF files into/from OCCT. Each operation needs configuration node.

Providers grouped by Vendor name and Format type. The Vendor name is "OCC" The Format type is "GLTF" The import process is supported. The export process is supported.
