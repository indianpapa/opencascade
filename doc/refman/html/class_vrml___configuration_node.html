Detailed Description

The purpose of this class is to configure the transfer process for VRML format Stores the necessary settings for Vrml_Provider. Configures and creates special provider to transfer VRML files.

Nodes grouped by Vendor name and Format type. The Vendor name is "OCC" The Format type is "VRML" The supported CAD extensions are ".vrml", ".wrl" The import process is supported. The export process is supported.
